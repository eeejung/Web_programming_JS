산술연산자, 비교연산자, 논리연산자(!, &&, ||)
javascript엔진은 연산자에 따라 자동 형변환이 필요하면 형변환 수행합니다
javascript 코딩 스타일로 true일만 실행 또는 false일때만 실행 &&, ||로 처리합니다.
숫자로 구성된 문자열을 number로 형변환 방법 : Number() 또는 parseInt() 사용
조건1 && 조건2 는 조건1이 true이면 조건2을 실행시켜야 최종적으로 true, false결정
조건1 || 조건2 는 조건1이 false 이면 조건2을 실행시켜야 최종적으로 true, false결정
&&, || 연산을 단순 조건 if 문 대신 사용하는 예) React에서 {}
instanceof는 [인스턴스 instanceof 타입|부모타입 ] is a 관계에 대해 false, true리턴
?? 연산자는 null 또는 undefined일 때만 기본값을 설정할 수 있는 연산자 (ES11)
?. 인스턴스(객체).속성이 null 또는 undefined이어도 안전하게 접근 가능 (ES11)
Boolean() 또는 !! 를 사용해서 boolean으로 형변환 할 수 있습니다.
