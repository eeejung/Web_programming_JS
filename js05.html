Q> 배열의 요소 개수보다 많은 변수를 분해 할당에 선언하면?
A> 배열의 요소 개수보다 많은 변수를 선언하면, 초과된 변수에는 undefined가 할당됩니다.
        let nums = [100, 99, 98];
        let [a, b, c, d] = nums;
        console.log(a, b, c, d); // Output: 100, 99, 98, undefined
Q> 객체의 요소를 분해할당할때 객체에 존재하지 않는 key를 분해 할당 변수로 선언하면?
A> 객체에 존재하지 않는 key를 분해 할당 변수로 선언하면, 해당 변수에는 undefined가 할당됩니다.
        let user = { userid: 'tester', username: '테스터' };
        const { userid, username, userpwd } = user;
        console.log(userid, username, userpwd); // Output: tester, 테스터, undefined
