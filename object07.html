ES6에서 추가된 Map(key, value로 저장하는 자료구조) , Set (value 저장, 중복 허용하지 않음)
Map에 저장되는 key는 unique 해야 하며, 배열, 객체, 함수를 key로 사용할 수 있습니다.
Map에 저장된 순서대로 요소들을 순회합니다.
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map
Set도 저장한 순서대로 순회합니다
add(), has(), delete(), clear(), size, forEach(callback) 메서드 제공
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Set
ES6에서 함수 또는 클래스를 파일 단위로 모듈화하여 관리할 수 있는 기능 추가됨
export와 import 키워드를 사용해서 파일의 저장된 기능을 내보내기, 가져오기를 합니다
(함수, 변수, 객체 내보내기 export로 가능) -> 코드의 재사용성 + 유지보수성 향상 시킴
브라우저의 html 페이지에서 모듈을 사용하려면 script 태그의 type 속성에 module이라고 선언
모듈 파일의 확장자는 .js로 사용
모듈은 import를 사용해서 로드될 때 비동기 방식으로 로드됨 (pormise 객체)
실행중에 동적으로 모듈을 로드할때 import()를 사용하면 Promise 객체가 반환됩니다.

export default 를 사용하면 모듈에서 하나의 기본 값 또는 객체 또는 함수를 내보낼 수 있습니다.
import 할때 {} 중괄혼 없이 가져오기 할 수 있습니다.
