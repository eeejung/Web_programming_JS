javascript는 Prototype 기반의 언어입니다
객체 : 속성과 기능을 캡슐화
객체의 종류 :
Predefined Object : 내장 객체 (Object, Math, Date, Array, Number, Boolean,...), BOM객체(window, locator, navigator, screen, document), DOM(Node, document, Element,...)객체
Custom Object 정의 방법 : JSON, 함수문법을 사용해서 생성자 함수, ES6 부터 지원하는 class 문법
JSON 객체는 정적인 데이터 표현에 적합
예: mongodb에서 json -> bson
JSON객체는 동일한 구조의 객체를 여러 번 생성하려면 매번 수작업으로 정의해야 합니다. (인스턴스를 생성할 수 있는 틀이 없습니다), 재사용성이 낮음
JSON객체는 [[Prototype]] 기반의 상속이나 메서드 재사용 등 고급 기능을 사용 못함
동적으로 요소 추가 , 삭제 가능
JSON 문법으로 생성한 객체는 Object를 Prototype으로 하는 인스턴스 입니다
